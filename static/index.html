This website is a demo for my natural language date parser

What's new in this project

Input any sentence that you may add to your todo list, if it contains any date or relative date, the program will detect it for you

Example input: "Finish my homework before next friday". (say today is 2019 July 27)
The program will know that today is 2019/7/27 saturday, and next friday is 2019/8/2.

NOTE: lowercase only, don't add period at the end
